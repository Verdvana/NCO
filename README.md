# 数字控制振荡器

#### 代码介绍
    频率、相位可调的数字控制振荡器

* V1.0   2019.11.26
    * 输入信号位宽可定制；
    * 16位正余弦波输出；
    * 频率可调，理论上大于1小于时钟频率都可以，实际到时钟频率的十分之一以上波形就快不能看了；
    * 相位可调：0-100；
    * en使能加载相位控制字。
